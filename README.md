> React Router routing module with protected routes

Part of the [Modsynth](https://github.com/modsynth) ecosystem.

## Features

- React Router v6+ integration
- TypeScript support
- ✨ **ProtectedRoute Component**: Authentication-based routing (v0.2.0)
- ✨ **Navigation Hooks**: `useNavigate`, `useParams`, `useLocation` re-exports (v0.2.0)

## What's New in v0.2.0

- **ProtectedRoute**: Component for protecting authenticated routes
- **Navigation Hooks**: Convenient re-exports from react-router-dom

## Installation

```bash
npm install @modsynth/routing react-router-dom
```
